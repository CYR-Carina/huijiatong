# 绿色通道与勤工助学

## 一、绿色通道

### 1.1 政策说明
- 为切实保证家庭经济困难学生顺利入学
- 教育部规定各公办普通高等学校都必须建立"绿色通道"制度
- 即对被录取入学、家庭经济困难的新生，学校一律先办理入学手续

### 1.2 适用对象
- 家庭经济特别困难，无法按时缴纳学费和住宿费的新生
- 包括：建档立卡贫困家庭学生、低保家庭学生、特困供养学生等

### 1.3 办理时间
- 新生报到当天

### 1.4 办理流程
1. 新生在报到时向学院辅导员提出申请
2. 填写《绿色通道申请表》
3. 提交相关证明材料
4. 学院审核通过后，直接办理入学手续
5. 入学后1-2周内补办资助申请手续

### 1.5 注意事项
- 绿色通道只是缓交学费，不是免交
- 学生入学后需及时申请国家助学贷款或其他资助
- 学校会根据学生实际情况，制定还款计划

## 二、勤工助学

### 2.1 政策说明
- 学校组织学生利用课余时间，通过自己的劳动取得合法报酬
- 用于改善学习和生活条件的社会实践活动
- 勤工助学由学校统一组织和管理

### 2.2 岗位类型
- 校内岗位：图书馆助理、实验室助理、办公室助理等
- 校外岗位：经学校批准的校外合作单位岗位

### 2.3 工作时间
- 每周不超过8小时
- 每月不超过40小时
- 不影响正常学习

### 2.4 劳动报酬
- 由学校按月支付
- 校内岗位：每小时15-20元（以学校规定为准）
- 每月约300-800元

### 2.5 申请条件
- 家庭经济困难学生优先
- 学有余力，成绩合格
- 遵守校规校纪

### 2.6 申请时间
- 每学期开学后2周内

### 2.7 申请流程
1. 学生填写《勤工助学申请表》
2. 学院审核推荐
3. 学生资助管理中心统一安排岗位
4. 签订勤工助学协议
5. 上岗工作
