# 国家奖学金与国家励志奖学金

## 一、国家奖学金

### 1.1 资助对象
- 特别优秀的全日制普通本专科生（含高职生）、研究生

### 1.2 奖励标准
- 本科生：8000元/年（以当年官方政策为准）
- 研究生：硕士20000元/年，博士30000元/年（以当年官方政策为准）

### 1.3 申请条件（本科生）
- 热爱社会主义祖国，拥护中国共产党的领导
- 遵守宪法和法律，遵守学校规章制度
- 诚实守信，道德品质优良
- 在校期间学习成绩优异，社会实践、创新能力、综合素质等方面特别突出
- 学习成绩排名与综合考评成绩排名均位于年级或专业前10%

### 1.4 申请时间
- 每年9-10月

### 1.5 评审程序
1. 学生申请
2. 学院推荐
3. 学校评审
4. 公示无异议后上报
5. 教育部最终审批

## 二、国家励志奖学金

### 2.1 资助对象
- 品学兼优、家庭经济困难的全日制普通本专科生（含高职生）

### 2.2 奖励标准
- 5000元/年（以当年官方政策为准）

### 2.3 申请条件
- 热爱社会主义祖国，拥护中国共产党的领导
- 遵守宪法和法律，遵守学校规章制度
- 诚实守信，道德品质优良
- 在校期间学习成绩优秀
- 家庭经济困难，生活俭朴
- 已被认定为家庭经济困难学生

### 2.4 申请时间
- 每年9-10月

### 2.5 注意事项
- 同一学年内，申请国家励志奖学金的学生可以同时申请并获得国家助学金
- 但不能同时获得国家奖学金

## 三、两者区别

| 项目 | 国家奖学金 | 国家励志奖学金 |
|------|----------|--------------|
| 金额 | 8000元/年 | 5000元/年 |
| 对象 | 特别优秀学生 | 品学兼优+经济困难 |
| 成绩要求 | 前10% | 优秀 |
| 经济要求 | 无 | 家庭经济困难 |
