# 老年人健康科普手册

## 一、高血压管理

### 1.1 什么是高血压
- 收缩压（高压）≥140mmHg
- 或舒张压（低压）≥90mmHg
- 常见症状：头晕、头痛、耳鸣、失眠

### 1.2 日常注意事项
- 🔴 少吃盐：每天不超过5克（大约一啤酒瓶盖的量）
- 🔴 少吃油腻食物：肥肉、动物内脏要少吃
- 🔴 戒烟限酒：烟能戒就戒，酒尽量少喝
- 🟢 多吃蔬菜水果：每天至少半斤蔬菜
- 🟢 适量运动：每天散步30分钟，微微出汗就好
- 🟢 保持心情愉快：不要生气着急
- 🟢 定期测量血压：建议每周量2-3次

### 1.3 用药提醒
- 遵医嘱按时吃药，每天固定时间吃
- **不要自己停药或减药**，即使感觉好了也要继续吃
- 忘记吃药不要补双倍，按下一次时间吃就行
- 注意药物副作用，头晕、脚肿要及时告诉医生
- 定期去医院复诊，让医生调整药量

## 二、糖尿病管理

### 2.1 什么是糖尿病
- 血糖水平持续升高
- 常见症状：多饮、多食、多尿、体重减轻

### 2.2 日常注意事项
- 🔴 少吃甜食、含糖饮料：白糖、红糖、蜂蜜都要少吃
- 🔴 少吃油腻食物：油条、肥肉要少吃
- 🔴 控制主食量：每餐米饭不超过一碗
- 🔴 不要吃太多水果：选择苹果、柚子等含糖量低的
- 🟢 多吃蔬菜：特别是绿叶菜
- 🟢 规律运动：饭后散步30分钟
- 🟢 控制体重：不要太胖
- 🟢 定期测血糖：建议每周测2-3次

### 2.3 用药提醒
- 按时服药或注射胰岛素，不能自己停药
- **随身携带糖果或饼干**，防止低血糖
- 出现头晕、心慌、出冷汗时，可能是低血糖，赶紧吃颗糖
- 定期去医院复查血糖、检查并发症

## 三、日常养生建议

### 3.1 饮食原则
- 三餐定时定量
- 粗细搭配
- 营养均衡
- 细嚼慢咽

### 3.2 运动建议
- 选择适合自己的运动：散步、太极、广场舞
- 运动时间：上午9-10点，下午4-5点
- 运动强度：以微微出汗为宜
- 注意安全，避免跌倒

### 3.3 生活习惯
- 保证充足睡眠
- 不要熬夜
- 保持室内通风
- 注意个人卫生

## 四、常见不适处理

### 4.1 头晕
- 立即坐下或躺下休息
- 缓慢变换姿势
- 喝水
- 如果频繁发作，及时就医

### 4.2 胸闷气短
- 停止活动，休息
- 保持室内通风
- 解开衣领
- 如果持续不缓解，立即拨打120

### 4.3 关节疼痛
- 注意保暖
- 适当活动
- 避免过度劳累
- 必要时就医检查

- 不要相信"包治百病""祖传秘方"等虚假宣传

- 看病要带上社保卡，可以报销部分费用
- 慢性病患者要定期复查，不要只吃药不检查
- 如果医生让住院，不要为了省钱拖着不去
