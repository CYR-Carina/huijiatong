# 惠家通 Agent 启动说明

## 一、环境准备

### 1.1 安装依赖
```bash
# 进入项目目录
cd e:\惠家通

# 安装 Trae CLI（如果未安装）
npm install -g @trae/cli

# 初始化项目
trae init
```

### 1.2 配置环境变量
```bash
# 复制环境变量配置文件
copy .env.example .env

# 编辑 .env 文件，填写实际的 API Key
notepad .env
```

## 二、知识库配置

### 2.1 上传知识库文档
将 `kb/` 目录下的所有文档上传到 Trae 知识库：

```bash
# 上传资助政策库
trae kb upload --name "资助政策库" --path ./kb/ --pattern "*资助政策库*"

# 上传银龄关怀库
trae kb upload --name "银龄关怀库" --path ./kb/ --pattern "*银龄关怀库*"
```

### 2.2 验证知识库
```bash
# 查看知识库列表
trae kb list

# 查看知识库详情
trae kb show "资助政策库"
```

## 三、Agent 配置

### 3.1 导入 Agent 配置
```bash
# 导入核心配置
trae agent import --config ./agent/惠家通_Agent核心配置.yaml

# 验证配置
trae agent validate
```

### 3.2 配置 UI 界面
```bash
# 导入 UI 配置
trae ui import --config ./agent/UI界面描述.yaml
```

## 四、启动服务

### 4.1 开发模式启动
```bash
# 启动开发服务器
trae dev --port 8080
```

### 4.2 生产模式启动
```bash
# 构建项目
trae build

# 启动生产服务器
trae start --port 8080
```

### 4.3 验证服务
打开浏览器访问: http://localhost:8080

## 七、注意事项

1. **API Key 安全**: 请妥善保管您的 API Key，不要提交到代码仓库
2. **知识库更新**: 定期更新知识库文档以保持政策信息最新
3. **语音配置**: 首次使用需配置火山引擎 ASR/TTS 服务
4. **端口占用**: 如果 8080 端口被占用，请修改 .env 中的 PORT 配置
5. **网络环境**: 启动前请确保网络通畅，需要连接大模型和语音服务
