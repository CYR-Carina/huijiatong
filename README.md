# 🧠 惠家通 — 乡村家庭智能服务 Agent

> 一个 Agent，服务整个农村家庭——学生问资助，老人防诈骗，全家享便利。

## 项目简介

惠家通是一款面向乡村家庭的 AI 智能服务助手，基于火山引擎 Trae 平台 + 豆包大模型构建。通过语音/文字交互，为农村家庭提供资助政策咨询、诈骗识别劝阻、健康科普和养老政策查询等服务。

## 核心功能

| 功能 | 说明 |
|:-----|:-----|
| 💰 资助政策问答 | 助学贷款、助学金、奖学金、绿色通道、勤工助学 |
| 📋 条件自测 | 输入家庭情况 → 智能推荐匹配资助项目 |
| 🛡️ 诈骗识别劝阻 | 分析可疑电话/短信，判断是否为诈骗并警示 |
| 🏥 健康科普 | 高血压/糖尿病管理、日常养生建议 |
| 👴 养老政策查询 | 养老金、医保、高龄补贴、社保卡办理 |

## 技术架构

- **大模型**: 豆包大模型（温度0.3）
- **语音**: 火山引擎 ASR + TTS（慢速模式，适合老年人）
- **知识库**: 双知识库架构（资助政策库 + 银龄关怀库）
- **意图识别**: 关键词规则 + LLM 分类
- **前端**: 原生 HTML/CSS/JS，适老化大字版设计

## 知识库

### 资助政策库
- 国家助学贷款（生源地/校园地，最高20000元/年）
- 国家助学金（平均3300元/年）
- 国家奖学金（8000元/年）/ 励志奖学金（5000元/年）
- 绿色通道与勤工助学
- 常见问题FAQ（28个问答）

### 银龄关怀库
- 防诈骗指南（6类诈骗 + 防骗五要原则）
- 健康科普手册（高血压/糖尿病管理）
- 养老政策指南（养老金、医保、高龄津贴）
- 常见问题FAQ（20个问答）

## 快速开始

### 本地运行

```bash
# 进入项目目录
cd 惠家通

# 复制环境变量配置
cp .env.example .env
# 编辑 .env 填写 API Key

# 启动本地服务器
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

### 部署

支持部署到火山引擎 IGA Pages、Vercel、GitHub Pages 等平台。

## 安全边界

- 不编造政策，始终基于知识库回答
- 不提供医疗诊断，仅做健康科普
- 涉及金额标注"以当年官方政策为准"
- 诈骗类问题先警示再解答
- 不知道的问题引导联系官方渠道
